Шифр Цезаря

Каждая буква заменяется на букву, стоящую на фиксированное число позиций правее.

3
Результат:
KHOOR ZRUOG
Математика
E(x) = (x + k) mod 26
x — номер буквы (A=0...Z=25),
k — сдвиг.
H (7) + 3 = 10 → K

Шифр Виженера

Сдвиг для каждой буквы определяется ключевым словом.

Результат:
LXFOPVEFRNHR
Математика
E = (tᵢ + kᵢ) mod 26
tᵢ — буква текста,
kᵢ — буква ключа.
A(0) + L(11) = 11 → L

RSA (демо)

Безопасность основана на сложности разложения на множители.

Открытый ключ
(5, 323)
Закрытый ключ
(173, 323)
Результат:
42
Математика RSA
n = p×q
φ(n) = (p-1)(q-1)
e×d ≡ 1 mod φ(n)
17×19=323, φ=288
e=5, d=173
